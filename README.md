# Zomato Clone Project

A full-stack web application that replicates core features of Zomato, built with React and Node.js.

## Project Structure

```
Zomato/
├── frontend/          # React frontend application
└── backend/           # Node.js backend server
```

## Frontend

The frontend is built with React and includes:
- Modern UI components
- Responsive design
- State management
- API integration

### Running the Frontend

```bash
cd frontend
npm install
npm run dev
```

## Backend

The backend is built with Node.js and Express, providing:
- RESTful API endpoints
- Database integration
- Authentication
- Business logic

### Running the Backend

```bash
cd backend
npm install
npm start
```

## Features

- Restaurant listings
- Search functionality
- User authentication
- Reviews and ratings
- Order management

## Technologies Used

- Frontend: React, Vite, CSS
- Backend: Node.js, Express
- Database: MongoDB
- Authentication: JWT

## Getting Started

1. Clone the repository
2. Install dependencies for both frontend and backend
3. Set up environment variables
4. Run both frontend and backend servers
5. Access the application at `http://localhost:5173`
